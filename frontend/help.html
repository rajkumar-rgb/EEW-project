Help & Support

Frequently Asked Questions

1. How does the prediction system work?

The system uses machine learning to analyze seismic station P-wave arrival times and estimate the earthquake’s epicenter.

2. What data do I need to enter?

Enter station codes and their P-wave arrival times. The system predicts the epicenter location using trained ML models.

3. Why is prediction not accurate?

Accuracy depends on station network density and input quality. This is an educational demonstration system.

4. Can it detect real earthquakes?

No, this tool is for academic use only.

Troubleshooting

  • Verify input fields are filled correctly.
  • Ensure station codes appear in the dataset used to train the model.
  • Backend server must be running for predictions.
  • Check Tailwind CDN connectivity if styling looks broken.
