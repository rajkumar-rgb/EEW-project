Earthquake Dataset Information

Dataset Overview

The dataset used in this Earthquake Early Warning System comes from the Japan Meteorological Agency (JMA). It contains detailed earthquake event records including P-wave arrival times, hypocenter coordinates, and event metadata.

The data was cleaned, filtered, and standardized to build a machine-learning-ready dataset for training the epicenter prediction model.

Only high-quality events and reliable stations were kept to improve accuracy.

Data Processing Workflow

  • Extracted raw ZIP archives of annual JMA earthquake logs.
  • Converted large text logs into structured CSV format.
  • Removed incomplete or low-quality seismic readings.
  • Kept only stable, consistently reporting stations.
  • Normalized P-wave arrival times for ML model compatibility.
  • Produced ~16,000 high-quality events for training.
