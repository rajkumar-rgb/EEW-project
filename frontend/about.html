About Earthquake Early Warning System

The Earthquake Early Warning (EEW) System is a machine-learning powered platform designed to predict earthquake epicenters using real seismic P-wave arrival times. The dataset originates from the Japan Meteorological Agency (JMA), cleaned and refined for high prediction accuracy.

The purpose of this project is to demonstrate the potential of AI in providing rapid earthquake alerts before destructive shaking begins. This system is made for educational and research purposes.

Key Features

  • Accurate epicenter prediction using ML techniques.
  • Clean, high-quality seismic datasets from JMA.
  • Fast, user-friendly prediction UI.
  • Academic demonstration of early warning technology.

Project Motivation

Earthquakes cannot be prevented, but their effects can be reduced using fast alerts. This project shows how P-wave detection and machine learning can contribute to safer, more efficient early warning systems.

Technologies Used

Machine Learning

Random Forest, MultiOutput Regression, Feature Engineering.

Frontend

HTML, Tailwind CSS, Smooth UX.

Backend

FastAPI (Python), REST APIs, Model Hosting.

Dataset

JMA Earthquake Archives.
